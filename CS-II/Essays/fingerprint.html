Hashing assigns values to data sothe operator can easily find said data which is why Hashing is so useful and efficient.

SSL interception

SSL interception uses an algorithm that specifies what traffic to stop, block, or allow through. This cannot be prevented however it can be detected. One way to detect an SSL interception is using certification authroity.

false positive/negative

A false postive is when the system detects a problem, or an anomaly, but turns out to be nothing. A false negative is when the system is unaware of a problem while there is a problem currently happening in the system.
